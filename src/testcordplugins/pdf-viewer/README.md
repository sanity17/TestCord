# PdfViewer

Inline PDF previews for Discord attachments.

PDFs are downloaded through a small native helper (CORS otherwise blocks the renderer) and rendered with [pdf.js](https://mozilla.github.io/pdf.js/) loaded from cdnjs.

Thanks to [@semon009](https://github.com/@semon009) for the toolbar's Fit Width icon, the per-page hover badges, the resize grip indicator, and the updated Discord theme variables.
